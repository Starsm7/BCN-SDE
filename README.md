# BCN-SDE

This is a Pytorch implementation of BCN-SDE, a causal neural network of stochastic differential equation.

![](https://github.com/Starsm7/BCN-SDE/blob/main/visual/BCN-SDE.svg)

The schematic illustration of BCN-SDE algorithm. We have provided the code for implementing causal networks and cell type consistency, bidirectional SDEs, as well as the simulated dataset, respectively.
